# Student Management System

## Overview

This is a Full-Stack Web Application developed as a Final Capstone Project.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Features

- User Registration
- User Login
- JWT Authentication
- Add Student
- View Students
- Update Student
- Delete Student

## Project Structure

frontend/
backend/
docs/

## Run Backend

```bash
node src/server.js
```

## Run Frontend

```bash
npm start
```
